## Markdown 常用语法记录 + 栗子展示

### 🌰例子1 : 
# 这是一级标题
## 这是二级标题
### 这是三级标题

```
  # 这是一级标题
  ## 这是二级标题
  ### 这是三级标题
```

### 🌰例子2(字体) : 
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~

```
  **这是加粗的文字**
  *这是倾斜的文字*`
  ***这是斜体加粗的文字***
  ~~这是加删除线的文字~~
```

### 🌰例子3(引用) : 
>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容

```
  >这是引用的内容
  >>这是引用的内容
  >>>>>>>>>>这是引用的内容
```

### 🌰例子4(分割线) : 
---
----
***
*****

```
  ---
  ----
  ***
  *****
```
